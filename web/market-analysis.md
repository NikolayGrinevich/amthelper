# AmtHelper — анализ рынка и боли пользователей

_Дата: 2026-06-19_

## 1. Топ-15 болей иммигрантов в Германии

### Бюрократические процессы
1. **Terminvereinbarung (запись на приём)** — невозможно попасть к Ausländerbehörде/Finanzamt/Bürgeramt. В Берлине средний срок — 27–33+ дней, для Anmeldung иногда месяцы. Нет единой очереди, много перезагрузок сайта.
2. **Неизвестные дедлайны** — официальные письма (Bescheide) содержат неочевидные сроки; страх пропустить их и получить штраф/сокращение выплат.
3. **Анкеты на немецком** — в 2024–2025 всё ещё требуют Sprachkenntnisse B1, хотя многие приезжают с нулём; нигде нет упрощённых версий.
4. **Сбор документов** — неясно, что именно нужно; разные требования в каждом городе/ Bezirk.
5. **Потеря документов / заявлений** — если документ потеряли в пути в ведомство, приходится начинать заново; нет трекинга.

### Коммуникация с ведомствами
6. **Нечитаемые Bescheide** — громоздкие бюрократические формулировки, сложно понять, что именно решили и можно ли оспорить.
7. **Нет статуса заявки** — после подачи неясно, на какой стадии; приходится самому звонить/ездить.
8. **Неравное обращение** — отказ, сокращение, требование дополнительных бумаг без объяснения причин.
9. **Сложность переноса Termin** — если не можешь прийти, перенести приём часто невозможно, ведомства штрафуют за пропуск.
10. **Языковой барьер в кризисных ситуациях** — сancellation/sanction/Bescheid приходит на немецком, переводчик часто недоступен.

### Жизнь с помощью государства
11. **Jobcenter/Bürgergeld: Sanctions** — сокращение выплат на 10–100% за мелкие нарушения; нет понятной защиты.
12. **Квитанции и beweisfähige Nachweise** — требуется доказать доставку/получение; неясно, что считается доказательством.
13. **Krankenkasse: спорные Abrechnungen** — путаница с учётом, долгие разбирательства по возмещению.
14. **Семейные процедуры (Ehegattennachzug, Kindernachzug)** — отдельные требования, долгое ожидание.
15. **Потерянные уведомления** — письма часто приходят на старый адрес или с опозданием; нет цифрового шлюза, кроме ELSTER/Postident, который неохотно используется.

## 2. Конкуренты и их недостатки

| Продукт / Канал | Что делает | Недостатки |
|---|---|---|
| **Finanzamt.de / ELSTER** | Налоговые декларации | Только налоги, сложный интерфейс, только DE, нет ИИ |
| **Ankommen.de / BAMF App** | Информация для беженцев | Только новоприбывшие, нет генерации писем, нет Follow-up |
| **ALL-INKL / Expense Tracker** | Учёт расходов | Не бюрократия, только учёт |
| **PDF-Dokumenten-Scanner / OCR Apps** (Play Market) | Сканирование | Только OCR, нет анализа содержания, нет рекомендаций |
| **Google Translate / DeepL** | Перевод | Без контекста бюрократии, часто ошибается в терминах |
| **Jetzt.de / Expatica Guides** | Статьи и гайды | Только информация, нет action-инструментов |
| **Toytown Germany (закрыт)** | Форум / сообщество | ЗАКРЫТ — главный рынок-провал; нет аналога |
| **Berlin.de / Muenchen.de** | Единые порталы | Сложная навигация, только DE, нет персонального лога |
| **Welcome App Germany** | Онбординг | Только стартовые шаги, нет глубины по конкретным ведомствам |
| **Simple Germany / AllAboutBerlin** | Гайды на EN | Только тексты, нет интерактивных инструментов |

**Ключевой вывод:** Нет продукта, который объединяет анализ документа + генерацию оспорительного письма + трекинг дедлайнов + i18n.

## 3. Пробелы рынка

1. **Ассистент для конкретных шагов** — никто не делает "сказал, что упало на сайте → показал, что делать дальше".
2. **Мультиязычный мост** — русский/украинский/румынский почти не покрыт существующими приложениями.
3. **Документ-аware помощник** — загрузил Bescheid → получил пошаговую расшифровку и шаблоны ответов.
4. **Трекинг Termin** — единый календарь всех ожидаемых приёмов и дедлайнов.
5. **Сообщество закрылось** — Toytown Germany исчез, пользователи ищут замену. AmtHelper может стать this replacement + аддон.

## 4. Рекомендации по фичам и шаблонам

### Must-have (MVP)
- Анализ PDF: краткое содержание, выявление прав и дедлайнов.
- Генерация ответов: 15–20 шаблонов под Finanzamt, Jobcenter, Ausländerbehörде.
- Трекер: хранение ID документа, статуса, дедлайна.
- Категории шаблонов с поиском + фильтрами.
- i18n: DE, RU, UK, RO — с первого дня.

### High-value
- **Termin Radar** — мониторинг открытых слотов на Berlin.de / service.berlin.de + алерт.
- **Санкции-чекер** — если Jobcenter срезает выплаты, показать, можно ли оспорить.
- **Документ-менеджер** — загрузка + хранение всех Bescheide; статус "обработан / ожидает ответа".
- **Письма готовые к печати/отправке** — PDF с правильным оформлением.

### Nice-to-have
- **Созвон с экспертом** — интеграция с юристом/консультантом по бюрократии.
- **Групповые шаблоны** — для семейного воссоединения, студенческих групп.
- **Оценка риска** — ML-подсказка: высокая/средняя/низкая вероятность успеха апелляции.

## 5. Приоритизация

| Приоритет | Фича | Почему |
|---|---|---|
| P0 | Трекер + дедлайны | Страх упустить срок — основная боль |
| P0 | 15-20 шаблонов | 80% запросов покрываются этими разделами |
| P0 | Анализ PDF | 💡 USP — правильно расшифровать Bescheid |
| P1 | Termin Radar | Высокий спрос, но требует парсинга сайта |
| P1 | Оспоривание Sanction | Эмоционально заряжено, релевантно |
| P2 | Документ-менеджер | Повышает Retention |
| P2 | Эксперт-консультация | Монетизация, но требует QA |

## 6. Стратегия позиционирования

- **Официально, но дружелюбно** — дизайн не пугает, но вызывает доверие.
- **Мультиязычность как ключ** — прячем сложность бюрократии за простым интерфейсом.
- **Готов к кризису** — для человека, получившего Bescheid за 3 дня до дедлайна, AmtHelper даёт "следующий шаг", а не энциклопедию.
- **Ниша** — отсутствие Toytown Germany = возможность стать центром сообщества + инструментов.
